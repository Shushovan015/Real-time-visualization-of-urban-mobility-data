# Frontend

React + Vite client for the Urban Crowding Visualization project.

For full project documentation (problem statement, architecture, setup, screenshots), see the root README:
- [`../README.md`](../README.md)

## Local Run

```bash
cp .env.example .env
npm install
npm run dev
```

Default app URL: `http://localhost:5173`

## Environment Variables

- `VITE_API_TOKEN`
- `VITE_API_URL`
- `VITE_API_BASE_URL`
- `VITE_BACKEND_PROXY_TARGET`
